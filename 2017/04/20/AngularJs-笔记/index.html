推荐几个好用的angular的插件

angular-ui

官网: angular-ui

Github: angular-ui

如果写angular想用bootstrap的话，直接照着例子用，比较方便，而且官网给的例子还有plunker（可以在线调试、运行的网站，唯一不好的就是得翻墙）。github上很活跃，有14000+的星。

ngSweetAlert

Github: ngSweetAlert

oitozero这哥们把挺有名的alert插件sweetAlert给封装了一下，使用方法也很简单。（注：sweetAlert 这个插件很火，而且支持IE8，github上有15000+的星）

使用方法

loading-bar

官网: angular-loading-bar

Github: angular-loading-bar

这个插件是在发送get、post请求的时候，自动执行的。可以通过css修改样式（loading-bar.css），默认样式是左上角有个圈圈，页面上头有个读条。

使用方法

validation

官网: validation

Github: validation

这是做表单验证的时候用到的，可以自己写正则规则。需要注意在使用的时候，input必须有name，否则不验证。
